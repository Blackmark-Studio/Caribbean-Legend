Strength
{
强壮的海盗通常是优秀的先锋队员 和炮手。 
影响需要体力的技能, 尤其是"中型武器"、 "重型武器"和"加农炮"。 
~
决定每级生命值和最大物品携带容 量。 
}

Perception
{
感知敏锐的海盗可以成为神射手、 秘密探员和炮手。 
~
影响"精准度"和"潜行"技能。 
}

Agility
{
敏捷的海盗往往是先锋队员或水手 长的最佳人选, 登船作战是他们的专长。 
影响"轻型武器"、"中型武器"、"登船"、 "火器"。 
~
决定最大能量值。 
}

Charisma
{
有魅力的海盗是天生的领袖、谈判者和财务官。 
天生的领袖无需使用暴力就能从商 人那里获取货物, 他们会不战而降。 最优秀的水手会追随他, 每个港口最美丽的女人都会等待他 的到来。 
~
影响"领导力"和"交易"。 
决定可雇佣的军官数量 (A*2)。 
}

Intellect
{
聪明的海盗学习速度快, 职业选择广泛: 医生、 炮手和领航员。 
~
影响"交易"、 "加农炮"和"航海"
决定获取等级和特质的速度。 
}

Endurance
{
耐力强的海盗适合担任木匠和医生。 
~
影响"维修"和"防御"。 
与(P)一样, 决定最大生命值和最大物品携带容 量。 
}

Luck
{
幸运的海盗是完美的射手、 赌徒和冒险家。 
~
"火器"、 "机会主义"和"潜行"需要此属性。 
决定部分游戏事件的结果, 并对暴击几率有重大影响。 
}

Crit
{
基础暴击概率取决于装备物品、特质和属性的组合。 
~
(S)属性决定近战暴击几率。 
(S)和(I)属性决定射击暴击几率 - 但仅适用于高品质火器。 
~

敌人的装备和特质会改变你的暴击 几率。 
}

CharDefence
{
冷兵器/火器/敌人暴击的基础防御指标
~
表现受个人能力和装备影响
~
不随敌人能力和装备调整。 
}

FencingL
{
使用轻刃。 花剑等武器的技能。 
(R*0.9+I*0.1)
}

FencingS
{
使用中等重量。 均衡型武器 (如军刀) 的技能, 需要力量和敏捷。 
(P*0.6+R*0.4)
}

FencingH
{
使用重武器 (如大剑和斧头) 的技能, 需要大量体力。 
(P*0.9+E*0.1)
}

Pistol
{
使用各种火器的技能。 
(R*0.4+S*0.6)
}

Fortune
{
一点主动运气在赌博和掠夺中会很 有用。 
基于(S). 
}

Leadership
{
影响特定事件和对话的结果, 也是招募新船员的主要技能。 
(A*0.8+T*0.2)
}

Commerce
{
在特定交易对话中起关键作用, 影响买卖价格和船只管理总成本。 
(T*0.8+A*0.2)
}

Accuracy
{
这是航海最重要的技能之一。 优秀的船长不会浪费火药和炮弹去 驱散鱼群: 每轮齐射都将命中目标! 当咸涩的海风驱散舰桥上的硝烟时 , 地平线上将不再有敌船! 
(I*0.8+S*0.2)
}

Cannons
{
快速装填加农炮。 推炮就位, 当齐射的回声仍在舱壁间轰鸣时, 清膛、 装填、 瞄准、 浇水降温... 战斗白热化时不应有多余动作: 装填。 推炮。 开火! 
(T*0.6+P*0.4)
}

Sailing
{
在风暴和浓雾中引导船只的能力, 使用导航仪器和指挥舰队的技能。 即使是最大、 最可怕的船只, 在经验丰富的船长稳健的操控下也 会像鸟儿一样轻盈。 
取决于感知和学习能力 (I*0.2+T*0.8)
~
指挥特定等级的船只需要相应的最 低航海等级: 
1级船需要95
2级船需要80
3级船需要65
4级船需要45
5级船需要25
6级船需要05
航海等级不足会对所有技能和英雄 属性造成惩罚。 当前航海值与要求值的差距越大, 惩罚越强。 
}

Repair
{
当敌人用炮弹轰击你的船只, 劈裂船舷、 将船帆射成筛子; 当船体在风暴中开裂, 狂风将索具逐根撕裂时, 维修成为最重要的航海技能之一。 
(E*0.8+I*0.2)
}

Grappling
{
将船只与敌船对齐。 抛出钩子登船、 清扫甲板并夺取船只作为应得的战 利品 - 这需要极大的敏捷和经验, 但战利品肯定能补偿所有努力。 
(R*0.7+T*0.3)
}

Defence
{
日复一日, 船员逐渐与船只融为一体。 经验丰富的水手价值相当于十几个 酒馆里招募的新手。 每个好船长都必须保护自己的船员 , 不让他们愚蠢地送命。 
(E*0.5+A*0.5)
}

Sneak
{
在敌人面前悄无声息地通过。 在公海被追击时逃脱, 或瞬间凭空出现攻击毫无防备的敌 人 - 每个船长都必须具备适当的潜行技 能。 
(S*0.5+I*0.5)
}

Loyality
{
你的行为会影响军官的忠诚度。 正直的骑士会拒绝为恶棍效力, 而天生的强盗也不会长期保持忠诚。 
~
高权威值可以显著降低决策对军官 忠诚度下降的影响, 即使这些行为与他们的世界观不符。 
}

Authority
{
权威。 名望和荣誉使主角
与游戏中所有其他角色区分开来。 
~
权威是周围人 (首先是主角舰队中船只的船员) 对英雄的评价。 如果船长的权威严重下降, 他会被标记, 并被要求自愿或强制辞职。 如果权威高, 船员会团结在指挥官周围, 满怀信心地经受任何考验! 
~
当英雄拯救世界及其居民时, 荣誉会增长。 或者因违背承诺。 施暴和恶行而坠入最丑陋的深渊。 荣誉取决于角色的行为, 并反过来影响游戏中其他人对他的 看法。 
~
然而, 如果英雄过着单调乏味的生活, 选择风险最小的解决方案, 独自在空旷的海域航行, 他的荣誉和名望会像漏勺中的沙子 一样下降, 最终沦为中立普通的状态。 在游戏中, 社区的记忆是短暂的, 不超过游戏内半年。 
}

Abilities_available
{
该数字表示未分配的特质点数。 
}

Abilities_active
{
该数字表示角色已激活的特质。 
}

Rank
{
等级表示角色在游戏中的进度水平。 随着英雄经验的增加而提升。 随着英雄等级的提升, 其生命值也会根据耐力值增加相应 数值。 
~
随着英雄等级的提升, 他会遇到更强的对手, 并更频繁地找到更好的物品。 
}

Life
{
生命值为零意味着死亡。 一点生命值意味着无限可能。 
~
(E)决定每升一级获得的额外生命值。 
}

Health
{
活力是表示角色身体状态的指标。 每次在短时间内受重伤时活力会下 降。 如果避免战斗让伤口适当愈合, 活力会恢复。 如果置之不理, 会严重削弱属性和技能。 
}

Energy
{
能量是决定英雄近战战斗耐力的重 要参数, 每次攻击都会消耗能量。 简单地说, 参与战斗会使角色疲劳, 需要休息来恢复耐力。 
~
(R)决定你的最大能量值。 
}

Money
{
金钱是万恶之源。 皮亚斯特.泰勒.基尼... 如果都是流动资金, 有什么区别! 
}

Honor
{
荣誉是其他角色对你的态度的衡量 标准。 荣誉可以是高 (正面)、中立和低 (负面)。 荣誉根据你的行为而变化, 并决定游戏中其他角色如何看待你。 
~
英雄的名望会随着时间流逝而褪色 , 他的成就或失败会从人们的记忆中 抹去。 恶行不像善行那样容易被遗忘。 如果一个英雄不选边站或不执行任 何重要行动, 他的声誉最终会变成"未知的冒险家"。 
}

Weight
{
角色在不失去移动速度和不受其他 技能惩罚的情况下可以携带的最大 重量。 
取决于(P)和(E). 
}

Title
{
头衔表示角色对其服务国家的有用 程度。 
}

NextExp
{
经验阈值表示当前技能。 等级和能力分配了多少点, 以及获得另一点需要多少经验。 
~
技能在执行相应动作时发展。 例如, 每次准确的手枪射击都会为"火器"技能获得经验点。 这些技能点计入等级经验和个人/船只能力 (取决于技能类型)。 
~
跨越等级经验阈值后, 英雄将获得新等级, 并随之获得更多生命值和能力点, 玩家可以根据自己的选择分配。 
}

英格兰。 不列颠真正"统治海洋"还有一个世纪, 但如果说有一个崛起的海军力量可 以与西班牙的强大抗衡 - 那就是英格兰。 当法国在欧洲攻击西班牙势力时, 英国私掠船和海军巡逻队以无保护 的西班牙航运为食, 最终目标是主宰加勒比海。

France_descr
{
一百多年来困扰法国的天主教和胡 格诺派之间的宗教战争, 随着十五岁的路易十四.波旁登基而基本结束。 路易国王距离未来领导法国进入黄 金时代的"太阳王"还有很长的路要走, 他希望延续伟大的黎塞留红衣主教 的传统, 通过大力改革来加强国家机器。 欧洲仍有许多冲突, 但在路易国王眼中, 最大的威胁是波旁家族的死敌哈布 斯堡家族 - 目前统治着西班牙和神圣罗马帝国。 为了在国内外与西班牙竞争, 路易国王命令让-巴蒂斯特.柯尔贝尔重振法国海军。 新的高效税收制度使法国不仅能够 偿还近五十年战争的债务, 还能为海军和陆军改革拨款。 虽然法国陆军还不像奥利弗.克伦威尔的英国新模范军那样现代 化或身经百战, 但他们正在迅速成为欧洲大陆装备 最好。 训练最有素的线性战争军队之一。 土伦和圣纳泽尔正在建造最新设计 的新船, 内陆工厂正在铸造现代加农炮。 
~
与英国不同, 法国的主要关注点在欧洲大陆, 殖民事务往往次于保卫法国边境免 受哈布斯堡家族在奥地利。 意大利和西班牙的压力。 
~
由于这种忽视, 法国美洲公司决定通过法国骑士兼 总督菲利普.德.潘西促成的交易, 将其在加勒比安的列斯群岛的部分 殖民地出售给马耳他骑士团。 德.潘西和骑士团大团长让.德.拉斯卡里斯-卡斯泰拉尔希望通过这次购买, 通过教廷不赞成的更... 创造性的方法来弥补骑士团在地中 海追逐巴巴里海盗所损失的收入。 一位名叫米歇尔.德.蒙佩尔的有前途的骑士甚至提议对 西班牙航运进行私掠! 
~
因此, 法国贵族的精英们纷纷搭乘下一班 船前往加勒比殖民地, 希望在那里找到名望。 财富。 冒险, 并可能在此过程中扩张法兰西帝国。 
~
}

Holland_descr
{
荷兰。 荷兰联省是一个年轻的国家, 最近刚从与西班牙的八十年独立战 争中解放出来。 荷兰人是节俭勤劳的民族, 渴望贸易优势。 目前荷兰暗中帮助法国对抗共同敌 人 - 西班牙; 但总体而言, 寻求成为可以与任何客户交易 (或欺骗) 的中立国家! 
}

Spain_descr
{
西班牙。 一个伟大但正在衰落的殖民大国。 目前控制着加勒比群岛和西班牙大 陆的大部分地区, 西班牙发现自己正面临来自英国和 法国的持续压力, 他们试图为自己夺取新世界的无尽 财富。 
}

Pirate_descr
{
海盗。 这些人悬挂所有旗帜航行, 遵循海岸兄弟会的海盗法典 (尽管更像是指导方针)。 这并不妨碍他们在皇家港和托尔图 加等英法殖民地出售战利品, 那里的总督会睁一只眼闭一只眼。 根据情况, 海盗可能会为任何能给他们颁发私 掠许可证的势力"合法"战斗, 但最终他们的奖品与英法私掠船相 同 - 一艘漂亮的西班牙珍宝船。 
~
对任何海军巡逻队来说, 地平线上的海盗旗就像把培根扔给 饥饿的狼。 对于绝大多数海盗来说, 只有两种结局 - 绞刑架或刽子手的绞索。 
}

~
可以通过为特定国家做好事甚至行 贿来降低悬赏金额。 通过足够的努力, 可以恢复中立关系。 
}

Prisoner {囚犯}

Prisoner_desc
{
囚犯是被俘虏并违背意愿关押在你 船上的角色。 任何囚犯都可以在你船的货舱中接 受审问。 大多数时候, 你可以为他获得赎金, 如果是海盗, 还可以将他招募到英雄的船员中。 你可以囚禁你承诺运送的乘客, 当他因长途旅行而开始不满时。 船长是那些在登船时向你投降或在 公海乘船时被你俘虏的人。 同一国家的总督会给出最高的赎金 , 但赎金会波动, 总督们对囚犯价值的看法也是如此。

Partition_hint
{
必须不迟于每月1日向船员支付份额。 进行以下支付: 
~
每月维护费 - 水手和军官的工资。 受伤人员的补偿以及船只和设备的 维护费。 
~
战利品份额 - 分割掠夺的贵重物品并在船员成员 之间分配份额。 
~
所有捕获的船只和货物都需分享。 根据海岸兄弟会的法律, 已故水手或军官的份额归其亲属和 朋友所有。 
~
船员成员的份额如下: 
- 水手 - 1份
- 军官 - 10份
- 同伴 - 30份
- 船长 - 200-650份
此外, 船长作为其舰队船只的所有者, 还从每艘船获得20-120份 (取决于船只等级) , 并承担与船只维护和可能损失相关 的所有费用。 
如果船长持有专利, 战利品的10%将支付给国家财政部。 
~
延迟支付当月款项将导致船员在月 底前发生动乱, 而提前支付将提高他们的士气。 
~
延迟支付过去月份的款项将显著降 低船员的士气和军官及同伴的忠诚 度。 
~
}

{

~
探索模式适合初学者, 以及希望有更多空闲时间完成各种 支线任务和自由玩法的玩家。 在此模式下, 已移除开始主要任务线的等级限制 , 没有多种选择的情况下计时器被禁 用, 并且根据所选难度级别, 全局地图上的时间会变慢。 主要任务中的敌人变得不那么强大 , 主角船只在全局地图上的速度提高 5%.
