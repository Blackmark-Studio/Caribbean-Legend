Strength
{
强壮的海盗通常是优秀的先锋队员 和炮手。 影响需要体力的技能, 尤其是"中型武器"、 "重型武器"和"加农炮"。 ~决定每级生命值和最大物品携带容 量。
}
Perception
{
感知敏锐的海盗可以成为神射手、 秘密探员和炮手。 ~影响"精准度"和"潜行"技能。
}
Agility {动作协调与手部灵巧。~ 影响: “登船战”、“轻型武器”、“中型武器”、“手枪与火枪”, 以及每级获得的能量值。}
Charisma
{
有魅力的海盗是天生的领袖、 谈判者和财务官。 天生的领袖无需使用暴力就能从商 人那里获取货物, 他们会不战而降。 最优秀的水手会追随他, 每个港口最美丽的女人都会等待他 的到来。 ~影响"领导力"和"交易"。 决定可雇佣的副官的数量 (A*2)。
}
Intellect {与生俱来的倾向与天资、 机智以及迅速适应环境的能力。 ~ 影响“贸易”、“炮术”、“航海”和每级获得的能量值。}
Endurance
{
耐力强的海盗适合担任木匠和医生。 ~影响"维修"和"防御"。 与(P)一样, 决定最大生命值和最大物品携带容 量。
}
Luck
{
幸运的海盗是完美的射手、 赌徒和冒险家。 ~"火器"、 "机会主义"和"潜行"需要此属性。 决定部分游戏事件的结果, 并对暴击几率有重大影响。
}
Crit
{
对敌人造成致命一击或射击的基础几率。~取决于 S 和 P。~特长和装备可以提高此几率。
}
CharDefence
{
基础伤害防御属性。 按特定百分比降低所有受到的伤害。 ~ 这些数值受个人特长和装备影响。
}
FencingL
{
使用轻型剑、花剑等武器的技巧。 (R*0.9+I*0.1) ~ 影响武器伤害和攻击能量消耗。
}
FencingS
{
使用中型、平衡性良好的武器 (如军刀) 的技巧, 这类武器需要力量和敏捷。 (P*0.6+R*0.4) ~ 影响武器伤害和攻击能量消耗。
}
FencingH
{
使用重型武器 (如阔剑和斧头) 的技巧。需要极强的体力。 (P*0.9+E*0.1) ~ 影响武器伤害和攻击能量消耗。
}
Pistol
{
使用各种火器的技能。 (R*0.4+S*0.6) ~ 影响伤害和装填速度。
}
Fortune
{
一点主动运气在赌博和掠夺中会很 有用。 基于(S).
}
Leadership
{
影响特定事件和对话的结果, 也是招募新船员的主要技能。 (A*0.8+T*0.2)
}
Commerce
{
在特定交易对话中起关键作用, 影响买卖价格和船只管理总成本。 (T*0.8+A*0.2)
}
Accuracy
{
这是航海最重要的技能之一。 优秀的船长不会浪费火药和炮弹去 驱散鱼群: 每轮齐射都将命中目标! 当咸涩的海风驱散舰桥上的硝烟时 , 地平线上将不再有敌船! (I*0.8+S*0.2)
}
Cannons
{
快速装填加农炮。 推炮就位, 当齐射的回声仍在舱壁间轰鸣时, 清膛、 装填、 瞄准、 浇水降温... 战斗白热化时不应有多余动作: 装填、 推炮、 开火! (T*0.6+P*0.4)
}
Sailing
{
在任何条件下驾驶船只、 使用航海仪器并指挥舰队的能力。 即便是最任性最笨拙的船也会服从经验丰富的船长稳健的掌舵。 依赖于 '感知' 和 '天赋' (I0.2 + T0.8) ~ 要驾驭特定等级的船只, 需要相应的最低导航等级:  等级 1 要求 90 等级 2 要求 70 等级 3 要求 55 等级 4 要求 40 等级 5 要求 20 等级 6 要求 01 ~ 导航不足会导致英雄所有技能和属性受到惩罚。 当前导航值与所需值之间的差距越大, 惩罚越严重。
}
Repair
{
当敌人用炮弹轰击你的船只, 劈裂船舷、 将船帆射成筛子; 当船体在风暴中开裂, 狂风将索具逐根撕裂时, 维修成为最重要的航海技能之一。 (E*0.8+I*0.2)
}
Grappling
{
将船只与敌舰对齐, 投掷钩索登上颤抖的敌船, 清扫甲板并夺取战利品 — 这需要极高的敏捷和经验, 但战利品一定能弥补所有的努力。 (R*0.7+T*0.3) ~ 影响登船距离。
}
Defence
{
日复一日, 船员逐渐与船只融为一体。 经验丰富的水手价值相当于十几个 酒馆里招募的新手。 每个好船长都必须保护自己的船员 , 不让他们愚蠢地送命。 (E*0.5+A*0.5)
}
Sneak
{
在敌人面前悄无声息地通过。 在公海被追击时逃脱, 或瞬间凭空出现攻击毫无防备的敌 人 - 每个船长都必须具备适当的潜行技 能。 (S*0.5+I*0.5)
}
Loyality
{
你的行为会影响副官的忠诚度。 正直的骑士会拒绝为恶棍效力, 而天生的强盗也不会长期保持忠诚。 ~高权威值可以显著降低决策对副官 忠诚度下降的影响, 即使这些行为与他们的世界观不符。
}
Authority
{
权威、 名望和荣誉使主角与游戏中所有其他角色区分开来。 ~权威是周围人 (首先是主角舰队中船只的船员) 对英雄的评价。 如果船长的权威严重下降, 他会被标记, 并被要求自愿或强制辞职。 如果权威高, 船员会团结在指挥官周围, 满怀信心地经受任何考验! ~当英雄拯救世界及其居民时, 荣誉会增长。 或者因违背承诺、 施暴和恶行而坠入最丑陋的深渊。 荣誉取决于角色的行为, 并反过来影响游戏中其他人对他的 看法。 ~然而, 如果英雄过着单调乏味的生活, 选择风险最小的解决方案, 独自在空旷的海域航行, 他的荣誉和名望会像漏勺中的沙子 一样下降, 最终沦为中立普通的状态。 在游戏中, 社区的记忆是短暂的, 不超过游戏内半年。
}

Personal_abilities
{
扩展角色个人能力的特长。 通过提升个人技能等级获得进度。 ~ 达到门槛后, 你将获得一个新的个人特长点。 不同的特长需要不同数量的点数。 通常, 最强的特长位于发展树的末端。
}
Ship_abilities
{
影响船舶管理的特长。 通过提升船舶技能等级获得进度。 ~ 达到门槛后, 你将获得一个新的船舶特长点。 不同的特长需要不同数量的点数。 通常, 最强的特长位于发展树的末端。 ~ 主角可从分配在船上职务且熟练的军官那里继承这些特长。
}
Abilities_available {该数字表示未分配的特质点数。}
Abilities_active {该数字表示角色已激活的特质。}
Rank
{
等级是角色在游戏中发展水平的指标。 通过提升技能等级来获得进度。 ~ 随着角色等级的提高, 生命值和能量都会增加, 并且更强的敌人和新的物品将更频繁地出现。
}
Life
{
HP — 角色对物理伤害的抵抗力指标。 零意味着死亡, 一意味着无限可能。 ~ 数值取决于 P 和 E, 并随阶级增长。
}
Health {活力是表示角色身体状态的指标。 每次在短时间内受重伤时活力会下 降。 如果避免战斗让伤口适当愈合, 活力会恢复。 如果置之不理, 会严重削弱属性和技能。}
Energy
{
能量 - 角色的体力储备。   ~   所有攻击动作以及在徒手格斗中的快速奔跑都需要消耗能量; 在战斗中角色会疲劳, 恢复体力需要休整。   ~   能量的数值取决于 R 和 T, 并随阶级增长。
}
Money {金钱是万恶之源。 皮阿斯特、 塔勒、 几尼... 如果都是流动资金, 有什么区别!}
Reputation
{
荣誉反映了社会对角色的看法。~你的处事方式会影响荣誉的变化。~荣誉总会趋向中立状态。
}
Weight
{
重量是角色可携带的最大货物总质量。 取决于力量、 耐力、 个人特长和激活的护符。 ~ 超重的角色移动更慢, 受到惩罚, 且无法使用快速旅行。
}
Title {头衔表示角色对其服务国家的有用 程度。}
NextExp
{
经验阈值表示当前技能、 等级和能力分配了多少点, 以及获得另一点需要多少经验。 ~技能在执行相应动作时提升。 例如, 每次准确的手枪射击都会为"火器"技能获得经验点。 这些技能点计入等级经验和个人/船只能力 (取决于技能类型)。 ~跨越等级经验阈值后, 英雄将获得新等级, 并随之获得更多生命值和能力点, 玩家可以根据自己的选择分配。
}
