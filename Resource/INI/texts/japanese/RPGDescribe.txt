Strength
{
屈強な海賊は、 しばしば優れた先鋒や砲手となる。
「中装武器」 「重装武器」 「大砲」 など、 体力を要する技能に影響する。
~
レベルごとのHPと最大所持重量を決定する。
}
Perception
{
洞察力の高い海賊は、 狙撃手や密偵、 砲手にもなれる。
~
「命中精度」 と 「隠密」 スキルに影響。
}
Agility {動作の調整と手先の器用さ。~ 影響: 「白兵戦」 「軽装武器」 「中装武器」 「銃器」、 およびランクごとのエネルギー増加量。}
Charisma
{
カリスマ性のある海賊は、 生まれながらのリーダーであり、 交渉人、 そして優れた会計士です。
暴力に頼らずとも、 商人から品物を手に入れることができ、 相手は抵抗せずすべてを差し出すでしょう。 最高の船乗りが彼に従い、 どの港でも美しい女性たちが彼の到着を待ちわびます。
~
 「統率力」 と 「交易」 に影響します。
雇える士官の数は（A*2）で決まります。
}
Intellect {生まれ持った傾向や適性、 機転、 状況への素早い適応力。 ~ 「交易」 「砲術」 「航海」、 およびランクごとのエネルギー獲得量に影響。}
Endurance
{
屈強な海賊は船大工や船医として輝く。
~
「修理」と「防御」に影響。
(P)と同じく、最大HPと所持容量を決定。
}
Luck
{
運のいい海賊は射撃も賭け事も冒険もお手のもの。
~
「火器」 「好機」 「隠密」 の成長に必要な能力。
一部のイベント結果やクリティカル発生率に大きく影響する。
}
Crit
{
敵にクリティカルヒットやクリティカルショットを与える基本確率。
~
SとPに依存。
~
特典や装備でこの確率が上昇する。
}
CharDefence
{
基本ダメージ防御値。 受ける全ダメージを一定割合で軽減します。
~
この値は個人のパークや装備の影響を受けます。
}

Commerce
{
特定の取引会話で重要な役割を果たし、 売買価格や船の運営コスト全体に影響します。
(T*0.8+A*0.2)
}
Accuracy
{
大海原を行く者に欠かせない重要な技能のひとつ。 優れた船長は火薬も砲弾も無駄にせず、 一斉射撃は必ず的中させる！ 塩風が艦橋の煙を吹き払う頃、 水平線に敵船の姿はもうないだろう。
(I*0.8+S*0.2)
}
Cannons
{
大砲を素早く装填し、 押し出して、 砲声の残響が隔壁に響く間に、 掃除して再装填、 照準、 水をかけて過熱を防ぐ……戦闘の最中に余計な動きは無用。 装填、 押し出し、 発射！
(T*0.6+P*0.4)
}
Sailing
{
あらゆる状況下で船を操り、 計器を使い、 艦隊を指揮する能力。 どんな気難しく扱いにくい船も、 熟練の船長の確かな手には従う。
「洞察」と「才能」に依存（I0.2 + T0.8）
~
指定された等級の船を指揮するには、 対応する最低限の航海術レベルが必要：
 等級1は90必要
 等級2は70必要
 等級3は55必要
 等級4は40必要
 等級5は20必要
 等級6は01必要
~
航海術が不足していると、 すべてのスキルと属性にペナルティが発生する。 現在値と必要値の差が大きいほど、 ペナルティも重くなる。
}
Repair
{
敵が砲弾の雨を浴びせ、 船体を粉々にし、 帆をボロボロに裂くとき――嵐の中で船体がきしみ、 風が索具を一本ずつ引き裂くとき、 修理は最も重要な海の技術となる。
(E*0.8+I*0.2)
}
Grappling
{
敵船に自船をぴたりと寄せ、 フックを投げて震える敵に乗り込み、 甲板を駆け抜けて船を正当な戦利品として手に入れる――これには高い反応力と才能が求められるが、 手に入る戦利品はその苦労に十分見合うだろう。
(R*0.7+T*0.3)
~
白兵戦の開始距離に影響する。
}
Defence
{
日々、 乗組員は船と一体になっていく。 熟練の水夫は酒場の新米十人分の価値がある。 優れた船長は部下を守り、 無駄死にさせてはならない。  
(E*0.5+A*0.5)
}
Sneak
{
敵に見つからずにすり抜けたり、 広い海で追跡を振り切ったり、 不意に現れて油断した敵を襲う——どの船長にも必要な隠密の技術だ。
(S*0.5+I*0.5)
}
Loyality
{
あなたの行動は士官の忠誠心に影響します。 高潔な士官は悪党の下では働かず、 生まれつきの悪党は長く正直ではいられません。
~
高い権威があれば、 あなたの決断が士官の忠誠心に与える悪影響を大きく抑えることができます。 これは、 その行動が彼らの信念と食い違う場合に特に有効です。
}
Authority
{
権威、 名声、 そして名誉は、 主人公を他のすべてのキャラクターと隔てるものです。
~
権威は、 主人公が周囲（ 特に自分の艦隊の乗組員） からどう評価されているかを示します。 もし船長の権威が著しく低下すると、  「黒い印」 を突きつけられ、 自ら辞任するか、 強制的に降ろされることになります。 逆に権威が高ければ、 乗組員は指揮官のもとに団結し、 どんな試練も信頼をもって乗り越えるでしょう！
~
名誉は、 主人公が世界や人々を救うことで高まり、 約束を破ったり、 暴力や悪事を働くことで最も醜い底まで落ちていきます。 名誉はキャラクターの行動によって変化し、 それが他の登場人物からの評価にも影響します。
~
しかし、 主人公が単調で穏やかな日々を送り、 危険を避け、 誰とも関わらず孤独に海を旅するなら、 名誉も名声も手のひらの砂のようにこぼれ落ち、 やがて平凡で目立たない中庸へと沈んでいきます。 ゲーム内での世間の記憶は短く、 半年も持ちません。
}
Attention_abilities
{
注意！
一部のスキルは特定の主人公や士官では使用できません。
}
Personal_abilities
{
キャラクターの個人的な能力を広げるパークです。 個人スキルのレベルアップで進行度が上がります。
~
一定の進行度に達すると、 新たな個人パークポイントを獲得します。 パークごとに必要なポイント数は異なります。 通常、 最も強力なものはパークツリーの終盤にあります。
}
Ship_abilities
{
船の運用に影響するパーク。 船のスキルを上げることで進捗が得られます。
~
一定の進捗に達すると、 新たな船パークポイントを獲得。 パークごとに必要なポイント数は異なり、 最も強力なものはパークツリーの終盤にあります。
~
これらは、 船の役職に就く熟練の士官から主人公が引き継ぐことも可能です。
}
Abilities_available {数字は未割り当てのパークポイント数を示します。}
Abilities_active {数字はキャラクターが有効にしている特典の数を示します。}
Rank
{
ランクはキャラクターの成長を示す指標です。 スキルをレベルアップすると進行度が上がります。
~
ランクが上がると、 体力とエネルギーが増加し、 強力な敵や新しいアイテムがより頻繁に出現します。
}
Life
{
HP — キャラクターの物理的ダメージへの耐性を示す指標。 ゼロは死、 1は無限の可能性を意味する。
~
値はPとEに依存し、 ランクが上がるごとに増加する。
}
Health {体力はキャラクターの身体状態を示す指標です。 短時間に重傷を負うたびに減少します。 しばらく戦闘を避けて傷を癒せば回復します。 放置すると能力やスキルに大きなペナルティが発生します。}
Energy
{
エネルギー - キャラクターの体力の源。

~

近接戦での攻撃やスプリントはエネルギーを消費します。 戦闘中は疲労し、 回復には休息が必要です。

~

エネルギー値はRとTに依存し、 ランクが上がると増加します。
}
Money {金こそがすべてを支配する。 ピアストル、 ターラー、 ギニー……どれも現金に変わりはないさ！}
Reputation
{
名誉は社会がキャラクターをどう見るかを表します。
~
問題の解決方法によって変化します。
~
名誉は常に中立な状態に戻ろうとします。
}
Weight
{
重量は、 キャラクターが運べる貨物の最大合計質量です。
パワー、 エンデュランス、 個人のパーク、 アクティブなアミュレットに依存します。
~
過積載になると移動速度が低下し、 ペナルティを受け、 ファストトラベルができなくなります。
}
Title {称号は、 そのキャラクターが仕える国家にどれほど貢献しているかを示します。}
NextExp
{
経験値のしきい値は、 現在のスキル・ ランク・ アビリティに割り当てられたポイント数と、 次のポイント獲得に必要な経験値を示します。
~
スキルは対応する行動を取ることで成長します。 たとえば、 ピストルで正確に撃つたびに 「火器」 スキルの経験値が得られます。 これらのスキルポイントは、 ランク経験値や個人／ 船のアビリティ（ スキルの種類による） に加算されます。
~
ランク経験値のしきい値を超えると、 ヒーローは新しいランクを獲得し、 それに伴いライフとアビリティポイントが増加します。 これらのポイントはプレイヤーが自由に割り振ることができます。
}
TABLE_PERKS {ヒーロー能力}
TABLE_PERKS_desc
{
アビリティはキャラクターの追加パラメータで、 新たな利点を得たり、 これまで使えなかったゲーム機能を開放したりできます。 アビリティによって特定分野のエキスパートにも、 幅広く知識を持つ万能型にもなれます。
~
選択したアビリティをダブルクリックすると、 説明や取得条件（ ある場合） を確認できます。
~
一部のアビリティは、 前提となるアビリティを習得しないと選択できません。 まずツリー内の初期アビリティを習得し、 新しいアビリティを解放しましょう。
}
PERK_TABLE_NEED {必要な能力}
PERK_TABLE_NEED_desc {このアビリティを習得するには、 まずこの系統の他のすべてのアビリティを習得する必要があります。}
Nation_hint
{
ヒーローの所属国は、 ゲーム開始時に選択した国、 または現在発行されている特許の国（ 従軍中の場合） です。 所属国は 「キャラクター」 画面で確認できます。  「国」 画面では現在の船の旗が表示され、 正体を隠すために海上でのみ変更可能です。 ただし、 外洋の船や港の衛兵には偽装を見破られることがあります。 ヒーローの隠密が高いほど、 偽装の成功率が上がります。 自国から賞金首にされている場合、 自国にも敵と見なされます。
~
敵対認識は、 国家間の公式な関係や、 各国からかけられた賞金も考慮されます。
~
敵対都市ではクイックトラベルは利用できません。
}
Drunk_hint {酒はスキルに影響します。 酔っていると痛みは感じにくくなりますが、 繊細な作業はできなくなります。}
England_descr {イングランド。 ブリタニアが 「海を支配する」 時代はまだ一世紀先だが、 スペインの強大な海軍力に対抗しうる新興勢力があるとすれば、 それはイングランドだ。 フランスがヨーロッパでスペインの力を削ぐ一方、 イングランドの私掠船や艦隊は無防備なスペイン船を次々と襲い、 やがてカリブを支配することになる。}
France_descr
{
宗教改革以来百年以上にわたりフランスを苦しめてきたカトリックとユグノーの悲惨な宗教戦争は、 十五歳のルイ十四世ブルボンの即位によってほぼ終息した。 だが、 フランスを黄金時代へ導く“太陽王”となるにはまだ遠い若き王ルイは、 リシュリュー枢機卿の偉業を継ぎ、 積極的な改革で国家体制を強化しようと志している。 ヨーロッパには依然として多くの争いが残るが、 ルイ王の目に最大の敵として映るのは、 ブルボン家の宿敵ハプスブルク家――現在スペイン王と神聖ローマ皇帝の座にある一族だ。 スペインと欧州内外で競うため、 ルイ王はジャン＝バティスト・ コルベールにフランス海軍の再建を命じた。 新たな効率的な課税制度により、 フランスは五十年近い戦争で生じた負債を返済するだけでなく、 海軍と陸軍の改革資金も確保できるようになった。 フランス陸軍は、 オリバー・ クロムウェル率いるイングランド新模範軍ほど近代的でも歴戦でもないが、 新たな線形戦術のもとで急速に装備と訓練が進み、 欧州屈指の精鋭となりつつある。 トゥーロンやサン＝ナゼールでは最新設計の新造艦が建造され、 内陸の工場では近代的な大砲が鋳造されている。
~
イングランドとは異なり、 フランスの主な関心はヨーロッパ大陸にあり、 植民地政策はしばしばオーストリア・ イタリア・ スペインのハプスブルク家から国境を守るという差し迫った課題の前に後回しにされる。
~
こうした無関心のため、 フランス領アメリカ会社はカリブ海アンティル諸島の一部植民地を、 フランスの騎士で総督フィリップ・ ド・ ポワンシーの仲介により、 マルタ騎士団へ売却する決断を下した。 ポワンシーと騎士団総長ジャン・ ド・ ラスカリス＝カステラールは、 この買収によって、 地中海でバルバリア海賊を追い回して失った騎士団の収入を、 聖座が眉をひそめるような…より創意的な手段で補うことを目論んでいる。 有望な若き騎士ミシェル・ ド・ モンペールは、 スペイン船を標的に私掠活動を提案するほどだ！
~
こうしてフランス貴族の精鋭たちは、 名声・ 富・ 冒険、 そしてフランス帝国の拡大を夢見て、 次なる船でカリブ植民地へと旅立つことになる。
~
}

Pirate_descr
{
海賊たち——彼らはどの国旗にも従わず、  「コースト同盟」 の海賊規約（ もっとも、 あくまで目安だが） を頼りに生きている。 とはいえ、 ポート・ ロイヤルやトルトゥーガのようなイギリスやフランスの植民地では、 総督が見て見ぬふりをするおかげで、 略奪品の売買も難なくこなす。 状況次第では、 私掠免許を与えてくれる国のために 「正規の」 戦いをすることもあるが、 結局のところ、 イギリスやフランスの私掠船と同じく、 狙いはスペインの財宝船だ。
~
どんな海軍哨戒隊にとっても、 水平線に翻る海賊旗は飢えた狼に肉を投げるようなもの。 大半の海賊に待つ結末は、 絞首台か、 首吊り縄のどちらかだ。
}

WorldAlivePause {国家クエスト}

Prisoner {捕虜}
Prisoner_desc
{
捕虜は、 あなたの船で意に反して拘束されているキャラクターです。 どの捕虜も船倉で尋問できます。 多くの場合、 身代金を得ることができ、 相手が海賊なら英雄の乗組員として勧誘することも可能です。 目的地への長旅で不満を漏らし始めた乗客も、 拘束することができます。 キャプテンは、 白兵戦で降伏したり、 海上でボートにいるところを捕らえられた者たちです。 身代金の最高額は、 捕虜と同じ国の総督が支払いますが、 その額や捕虜の価値に対する総督の評価は変動します。
捕らえたキャプテンは、 さまざまな願いを主人公に訴えることがあります。 捕虜との対話を避けずにいれば、 思わぬ利益が得られるかもしれません。
}

Partition_hint
{
乗組員への分け前は毎月1日までに支払う必要があります。 支払い内容は以下の通りです。
~
月次維持費 - 船員と士官の合計給与、 負傷者への補償、 船や装備の維持費。
~
戦利品分配 - 奪った財宝を分け、 乗組員に割り当てること。
~
拿捕した船や貨物はすべて分配の対象です。 コースト同盟の規定により、 死亡した船員や士官の分け前は遺族や友人に渡されます。
~
乗組員の分け前は以下の通りです：
- 船員：1分け前
- 士官：10分け前
- コンパニオン：30分け前
- キャプテン：200～650分け前
さらに、 キャプテンは自分の艦隊の船の所有者として、 各船ごとに追加で20～120分け前（ 船の等級による） を受け取ります。 また、 船の維持や損失に関する全費用も負担します。
キャプテンが特許状を持つ場合、 戦利品の10%が国庫に納められます。
~
今月分の支払いが遅れると、 月末まで乗組員の不満が高まります。 逆に前払いすれば士気が上がります。
~
前月分の支払いが遅れると、 乗組員の士気と士官・ コンパニオンの忠誠心が大きく下がります。
~
}

CasualMode_desc
{
クラシックモードは、 主人公の物語に集中したいプレイヤー向けです。 探索モードよりやや難易度が高く、 より注意と努力が求められますが、 ゲーム作者が作り上げた世界観に深く没入できます。
~
探索モードは初心者や、 サイドクエストや自由なプレイに多くの時間を使いたいプレイヤーに適しています。 このモードではメインクエスト開始時のランク制限がなくなり、 選択肢の少ない場面ではタイマーが無効化され、 難易度に応じてワールドマップ上の時間経過が遅くなります。 メインクエストの敵は弱体化し、 主人公の船のワールドマップ移動速度が5%上昇します。
}
