SeaBattle {海戦}
SeaBattle_descr
{
海戦へようこそ、キャプテン！
あなたの主な目標は、 生き残り、 敵船を沈めることです。
~
すべての船には 「船体」  「帆」  「乗組員」 の3つの主要ステータスがあります。 船体がゼロになると沈没します。 帆が損傷すると速度が落ち、 乗組員が減ると白兵戦がしやすくなり、 船の行動全体が遅くなります。
~
各ステータスには最適な弾薬があります。 乗組員には散弾、 帆やマストにはチェーンショット、 船体には爆弾が有効です。 砲弾は万能で最長射程を持ちます。 ダメージは弾薬の種類と砲の大きさの両方に依存し、 大きい砲ほど威力が上がります。
~
主な射撃モードは2つあります。 クイックショットは敵が射界に入った時に使えます。 序盤は命中率や威力が高くありませんが、 動きのある戦闘ができます。 エイムショットは照準ボタンで発動し、 側面から高精度で撃てます。 慣れが必要ですが、 習得する価値があります。
~
敵の砲撃を避け、 まず帆やマストを狙いましょう。 敵の脅威が大きく下がります。
~
各船にはリグによって風に対する最適な進路があり、 最大速度を出せます。 これはコンパス上の黒い矢印で示されます。 自分の船の強みを活かすことが、 海戦で勝利する鍵です。
}

Prologue_Amulet {護符}
Prologue_Amulet_descr
{
アミュレットとタリスマンはキャラクターを一時的に強化する強力なアイテムで、インベントリの専用スロットに装備されます。
~
タリスマンはプレイヤーが自作できますが、アミュレットは発見または購入のみ可能です。
~
ゲームにはWardsも存在し、非常に強力で効果時間は無制限、専用のインベントリスロットを使用します。
~
特に難易度の高いスキル判定や、海上・陸上での激しい戦闘に備えて、常に複数のセットをインベントリに用意しておきましょう。
~
注意：有効化後にアミュレットやタリスマンを外すと破壊されます。慎重に判断してください。
}

Prologue_AutoFire {海戦}
Prologue_AutoFire_descr
{
あなたの最初の海戦はCharles de Maureの想像の中で行われます。 心配せず、 自由に試してみましょう！
~
すべての船には 「船体」 「帆」 「乗組員」 の3つの主要ステータスがあります。 船体がゼロになると沈没します。 帆が損傷すると速度が落ち、 乗組員が減ると白兵戦や再装填、 帆の操作に影響します。
~
弾薬ごとに用途があります。 散弾は乗組員用、 チェーンショットは帆やマスト用、 爆弾は船体用です。 カノン砲弾は汎用で最長射程を持ちます。 ダメージは弾薬の種類と発射する砲の大きさで決まります。 大きい砲ほど威力が上がります。
~
主な射撃モードは2つ。 敵が射界に入るとクイックショットが使えます。 序盤は命中率や威力は高くありませんが、 素早い戦闘が可能です。
~
戦闘が遅く感じたら、 時間操作キーでスピードを上げましょう。
}
Prologue_AimingFire {精密射撃}
Prologue_AimingFire_descr
{
狙い撃ちは舷側から非常に正確に射撃できます。 慣れが必要ですが、 その価値はあります。
~
砲の円は、 実際の舷側砲の位置と弾道を示します。 中央の照準点に合わせるには少し時間がかかります。
~
発射時、 砲はその瞬間の円の位置に撃つので、 必要なら目標の動きを予測して狙いましょう。
~
敵船に照準を合わせるとターゲットにできます。 捕捉後は、 どの部位にも狙いを定めたり、 先読み射撃も可能です。
~
キャプテンや砲手のスキルが上がると、 砲の照準速度や発射速度が向上します。
~
照準を素早く動かすと、 砲手が混乱し、 普段より照準が遅くなります。
~
狙い撃ちの舷側射撃は、 遠距離から帆やマストなど重要部位を狙うのに最適です。
}

Prologue_FightAttack {攻撃}
Prologue_FightAttack_descr
{
カリブの伝説の戦闘は苛烈かつ迅速ですが、 今は安全です。
~
すべてのキャラクターが武器を持ち、 戦闘スキルを備えています。 武器が優れていて英雄が熟練しているほど、 与えるダメージも大きくなります。
~
攻撃と防御のアクションがいくつかあります。
~
まずは三つの基本攻撃から始めましょう。
~
斬撃。 素早く、 ダメージは小さめ、 消費エネルギーは中程度です。
~
速攻。 斬撃より遅いですが、 中ダメージで消費エネルギーは少なめです。
~
強攻撃。 遅いが強力で、 大ダメージを与え、 ブロックを破ることもできます。
~
これらの攻撃をすべて海賊で試してみましょう。
}

Prologue_FightDefence {戦闘防御}
Prologue_FightDefence_descr
{
いいぞ！次は防御だ。
~
ブロック。 重い攻撃以外を防げる。 ブロック中はゆっくり移動でき、 エネルギーは消費しない。 ただし強敵は通常攻撃でもブロックを破ることがあるので注意。
~
パリィ。 どんな攻撃もはじき、 敵のバランスとエネルギーを奪う。
~
後退。 位置を変えたり、 銃撃を避けたりできる。
~
パリィやブロックで海賊の攻撃を防いでみよう。
~
最後に—とどめを刺せ。
}

Prologue_Marksmanship {射撃}
Prologue_Marksmanship_descr
{
火器は近接武器よりも珍しいですが、 カリブでは撃たれる危険が常にあります。 油断しないように。
~
陸上でも海上と同じく、 射撃モードは2種類あります： クイックショットとエイムショットです。
~
クイックショットはボタンを押すだけ。 素早いですが、 威力は控えめです。
~
エイムショットは大ダメージを狙えますが、 無防備になります。 狙う時間が長いほど威力が上がり、 ヘッドショットならさらに高ダメージです。
~
エイムボタンを押し続けてこのモードに入り、 攻撃ボタンで発射します。 エイムを離すと発射せずに解除されます。
}
